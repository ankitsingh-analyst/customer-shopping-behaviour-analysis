# customer-shopping-behaviour-analysis
Customer shopping behavior analysis using Excel, SQL, and Power BI to identify sales trends, customer patterns, and business insights.

## Overview
This project analyzes customer shopping data to understand purchase patterns, category performance, and seasonal trends. The objective is to convert raw data into meaningful insights that support business decision-making.

## Objectives
- Analyze customer purchase behavior
- Identify top-performing product categories
- Study seasonal trends in sales
- Generate insights for customer retention and growth

## Tools and Technologies
- Microsoft Power BI
- MySQL
- Microsoft Excel

## Dataset
The dataset contains customer-level information such as:
- Age, Gender
- Product Category
- Purchase Amount
- Season
- Payment Method
- Subscription Status

## Workflow
1. Data collection from CSV file  
2. Data cleaning (handling null values, duplicates, formatting)  
3. Data analysis using SQL (aggregation, filtering, KPI calculation)  
4. Dashboard development in Power BI  
5. Insight generation and interpretation  

## Key Insights
- Certain product categories contribute the majority of revenue  
- Customer purchasing behavior varies by season  
- Different customer segments show different buying patterns  

## Business Recommendations
- Focus marketing on high-performing categories  
- Plan promotions based on seasonal trends  
- Improve customer retention strategies  

## Outcome
Developed an interactive dashboard to analyze customer behavior and support data-driven decisions.
